# Bimpatiens_MaternalBrain
This repository was created for documenting code associated a manuscript investigating neurodevelopmental and neuromolecular plasticity in B. impatiens (titled: Queen and worker bumble bee brains respond differently to maternal- and queen-like behavioral states).

#Species: Bombus impatiens

This repository includes markdown files, R scripts, and snake files for processing and analysis. These include volumetric and gene expression analyses. The files for generating those results are included.

Manuscript authors: Mallory A. Hagadorn (corresponding author), Priscila K. F. Santos, Makenna M. Johnson Bird, Kenneth P. White, Karlee
Eck, Anna C. Figgins, Carson Stoker, Thuy-Tien T. Lindsay, Erica J. Brus, James P. Strange,
Karen M. Kapheim (corresponding author)
